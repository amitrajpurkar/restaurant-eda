# SKILL: Data Visualization

## Purpose
Communicate insights clearly and honestly.

## Principles
- Clarity over aesthetics
- One insight per chart
- Avoid misleading scales

## Tools
- matplotlib / seaborn / plotly (as appropriate)

## Output Expectations
- Well-labeled plots
- Clear captions
- Actionable insights

## Constraints
- No decorative charts
- No unexplained visual choices
