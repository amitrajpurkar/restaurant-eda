# SKILL: Data Cleaning & Preprocessing

## Purpose
Prepare data for modeling in a safe, explainable way.

## Principles
- Every transformation must be explainable
- Preserve raw data
- Avoid irreversible transformations early

## Techniques
- Missing value handling (explicit strategy)
- Outlier treatment (documented)
- Type normalization
- Date/time standardization
- Categorical encoding

## Output Expectations
- Cleaned dataset
- Transformation rationale
- Reproducible preprocessing steps

## Constraints
- No data leakage
- No hard-coded thresholds without justification
